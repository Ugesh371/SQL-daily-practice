# Day 1: Hospital Database (DDL & DML Basics)

## Objectives
* Create a database and table schema.
* Perform table alterations (`ALTER`, `MODIFY`).
* Practice database cleanup safety patterns using `RENAME` and `DROP`.

## Schema
### Table: `patient`
* `PatientID` (INT)
* `PatientName` (VARCHAR)
* `Age` (INT)
* `Gender` (VARCHAR)
* `AdmissionDate` (DATE)
